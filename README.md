# vue-zip-demo

vue2+elementUi预览zip压缩包中的png图片，mp3音频，不支持加密zip

## Project setup

``` 
yarn install
```

### Compiles and hot-reloads for development

``` 
yarn run serve
```

### Compiles and minifies for production

``` 
yarn run build
```

### Run your tests


``` 
yarn run test
```

### Lints and fixes files

``` 
yarn run lint
```
